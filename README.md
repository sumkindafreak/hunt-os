# HUNT OS

> A modular ESP32 platform for immersive live-action games.

## What is HUNT OS?

HUNT OS is a distributed operating system built around ESP32 devices that allows organisers to run immersive games from a single command system.

The long-term goal is to support everything from zombie survival games to treasure hunts, sci-fi adventures, medieval quests and escape-room style experiences using the same firmware ecosystem.

## Core Components

- ESP32-C3 Player Nodes
- ESP32-S3 Base Nodes
- Relay Nodes (planned)
- Audio Nodes (planned)
- Sensor Nodes (planned)
- Lighting Nodes (planned)

## Major Features

- ESP-NOW networking
- OLED interfaces
- NeoPixel lighting
- DFPlayer audio
- Scene Manager
- Scenario Manager
- Mission system
- Offline Command Center WebUI
- Progressive Web App hosted by the Base Node
- Future OTA updates and scenario packages

## Design Philosophy

- Modular
- Offline first
- Expandable
- Easy to maintain
- Hardware abstraction through named pin definitions
- Designed for both hobbyists and commercial attractions

## Project Status

HUNT OS is currently under active development, with hardware, firmware and documentation evolving together.

Every subsystem is being designed with future expansion in mind.
