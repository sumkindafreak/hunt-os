# HUNT Command Center Sections v0.1

## Dashboard

Shows:

- Current scenario
- Current scene
- Game phase
- Game timer
- Connected players
- Connected Base Nodes
- Relay Nodes
- Audio Nodes
- Sensor Nodes
- Battery warnings
- Network health
- Emergency Stop

## Game Control

Actions:

- Start game
- Pause game
- Resume game
- End game
- Reset scenario
- Reset players
- Emergency Stop

## Player Control

Actions:

- Heal
- Infect
- Eliminate
- Revive
- Change team
- Send message
- Flash player LED
- Play player alert

## Base Control

Actions:

- Change role
- Load scene
- Change brightness
- Change audio volume
- Trigger activation
- Reboot Base

## Relay Control

Actions:

- Relay ON
- Relay OFF
- Relay pulse
- All relays OFF
- Scene-linked relay triggers

## Audio Control

Actions:

- Play track
- Stop
- Set volume
- Loop ambient
- Trigger effect
- Test speaker

## Lighting Control

Actions:

- Status NeoPixel colour
- Decor NeoPixel colour
- Brightness
- Animation
- Speed
- Test pattern

## Network

Shows:

- Device ID
- Device type
- Firmware version
- Last seen
- Battery
- Signal estimate
- Current role or state

## Developer Mode

Tools:

- ESP-NOW packet monitor
- Raw packet sender
- API test panel
- Memory usage
- Uptime
- GPIO test
- Audio test
- NeoPixel test
- Relay test
